ECICS Web Scraper (JavaScript)

Built a web scraper that gathers regulatory information on over a 100,000 chemical substances from this website. Scraper uses a headless browser (puppeteer) and node.js to interact elements on page a traverse to multiple pages on site. Scraper stores information into a Mongo database so that it can be exported or searched later.
